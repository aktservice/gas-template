# Gas Template Project

## Overview

This repository provides a **Gas Template** – a starter kit for building modern web applications using Vite, TypeScript, and a curated development setup. It includes pre‑configured tooling for linting, testing, building, and running a development server, enabling you to focus on writing application code rather than configuration.

## Features

- **Vite** powered fast dev server and bundling
- **TypeScript** support with strict type‑checking
- **Jest** testing framework with TypeScript integration
- **ESBuild** configuration for custom builds
- **Docker** support for containerised development and production builds
- **Prettier** and **EditorConfig** for consistent code style
- **GitHub Actions** ready‑to‑use CI pipeline (add your own workflow files)

## Getting Started

### Prerequisites

- **Node.js** (>= 18)
- **npm** (comes with Node.js) or **yarn**
- **Docker** (optional, for containerised workflows)
- **WSL2** on Windows (recommended) – the repository is located at `\\wsl.localhost\Ubuntu26\home\aktdev\agent\gas-template`.

### Installation

```bash
# Clone the repository
git clone <repository-url>
cd gas-template

# Install dependencies
npm ci   # uses package-lock.json for reproducible installs
```

### Development Server

```bash
npm run dev
```

The Vite dev server will start at `http://localhost:5173` with hot‑module replacement enabled.

### Building for Production

```bash
npm run build
```

The compiled assets will be emitted to the `dist/` directory.

### Running Tests

```bash
npm test
```

Jest will run all tests located under the `test/` directory.

## Scripts Overview

| Script | Description |
|--------|-------------|
| `dev` | Starts Vite development server |
| `build` | Builds the app for production |
| `test` | Runs Jest test suite |
| `lint` | Executes ESLint & Prettier checks |
| `docker:build` | Builds Docker image (requires Docker) |
| `docker:run` | Runs containerised app |

## Docker Usage (Optional)

To build a Docker image:

```bash
npm run docker:build
```

To run the container:

```bash
npm run docker:run
```
