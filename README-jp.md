# Gas テンプレートプロジェクト

## 概要

このリポジトリは **Gas テンプレート** を提供します。Vite、TypeScript、そして開発に必要なツールがあらかじめ設定されたスタートキットで、設定に時間を取られずにアプリケーション開発に集中できます。

## 特長

- **Vite** による高速開発サーバーとビルド
- **TypeScript** の厳格な型チェック
- **Jest** を用いたテストフレームワーク
- **ESBuild** を活用したカスタムビルド設定
- **Docker** を利用したコンテナ開発・本番ビルド
- **Prettier** と **EditorConfig** による統一されたコードスタイル
- **GitHub Actions** 用意済み（CI パイプラインはご自身で追加）

## はじめに

### 前提条件

- **Node.js** (>= 18)
- **npm**（Node.js に同梱）または **yarn**
- **Docker**（オプション、コンテナ利用時）
- **WSL2**（Windows 推奨） – 本リポジトリは `\\wsl.localhost\Ubuntu26\home\aktdev\agent\gas-template` にあります。

### インストール手順

```bash
# リポジトリをクローン
git clone <リポジトリURL>
cd gas-template

# 依存パッケージをインストール
npm ci   # package-lock.json を利用した再現性のあるインストール
```

### 開発サーバー起動

```bash
npm run dev
```

デフォルトで `http://localhost:5173` に Vite の開発サーバーが立ち上がり、ホットリロードが有効です。

### 本番ビルド

```bash
npm run build
```

ビルド成果物は `dist/` ディレクトリに出力されます。

### テスト実行

```bash
npm test
```

`test/` ディレクトリ以下の Jest テストが実行されます。

## スクリプト一覧

| スクリプト | 説明 |

| `dev` | Vite 開発サーバーを起動 |
